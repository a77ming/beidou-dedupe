# Project Code Map: video-dedupe-desktop

## What UI Is "Home" (首页到底是哪里的)

This app has **two UIs** by design:

1. **Main app UI (首页 / 主窗口)**
   - Dev: loaded from Vite dev server (`npm run dev`) into Electron main window.
   - Prod: loaded from the built file `renderer/index.html` (generated by `vite build`).
   - Source entry (Vite): `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/index.html`
   - React entry: `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/src/renderer/main.jsx`
   - Root component: `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/src/renderer/App.jsx`

2. **Publish UI (发布窗口壳 / 独立窗口)**
   - Purpose: provides a fixed toolbar and hosts a `BrowserView` pointing to the remote publish site.
   - File: `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/electron/publish_shell.html`

If you feel "UI 有好几套", it is mainly because:
- Main window uses **React renderer** (built into `renderer/` folder).
- Publish window uses **plain HTML shell** (toolbar) + `BrowserView` remote content.

## Runtime / Build Topology

- Electron main process:
  - Entry: `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/electron/main.js`
  - Creates:
    - `mainWindow` (the main React UI)
    - `publishWindow` + `publishView` (remote publish site, with toolbar shell)

- Preload bridge (main window):
  - Used by Electron main window: `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/electron/preload.cjs`
  - Exposes `window.api.*` methods to renderer.

- Vite build output:
  - Output dir: `renderer/` (ignored by git; produced by `npm run build`)
  - Config: `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/vite.config.js`

## Main Data Flow

1. Renderer (React UI) calls `window.api.*` (preload).
2. Electron main process handles IPC in `electron/main.js`:
   - File picking: `select-videos`, `select-output-dir`
   - Strategy loading: `load-strategies` (remote URL from env `STRATEGY_URL` or `config.json`)
   - Python runtime: `runtime-status`, `runtime-install`
   - Video processing: `run-dedupe` -> spawns python `tools/process_video.py`
   - Publish window: `open-publish-*`, `publish-files`, `close-publish`
3. Python processing:
   - Entrypoint script: `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/tools/process_video.py`
   - Python module root: `/Users/ming/.codex/worktrees/9607/video-dedupe-desktop/processor/core/video_processor.py`

## "Folder Cleanup" Rules Used Here

- `src/` is now the only place for **editable UI sources** (React).
- `renderer/` is the **build output** folder created by Vite (kept ignored by git).
- `electron/` contains only Electron main/preload and the publish shell.
- `processor/` + `tools/` contain the Python runtime and scripts.
